Every Claude session, I used to spend the first 10 minutes re-explaining my own project.

Now I just start working.

I built **recall** — a persistent session memory for Claude Code, Codex, and Claude Desktop. It indexes what happened in each session, loads the relevant context on startup, and lets you save + resume across the context limit without losing a thread.

Open source. MIT. Five minutes to install.

→ https://github.com/ashrocket/recall
